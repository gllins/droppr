# Droppr — Mini Rede Social em PHP

Droppr é uma mini rede social inspirada no Twitter, o projeto iniciou como trabalho da disciplina Web II no curso de Análise e Desenvolvimento de Sistemas, e posteriormente
foi ampliado e refinado para compor meu portifólio profissional.  
O objetivo é demonstrar habilidades em backend, CRUD completo, autenticação de usuários e integração com banco de dados MySQL, utilizando PHP puro.

---

## Sobre o Projeto
O Droppr permite que usuários criem contas, façam login, publiquem mensagens curtas (“drops”), sigam outros usuários, curtam posts, comentem e interajam dentro de um feed atualizado.  
É um projeto focado em lógica, organização e fluxo completo de uma aplicação web real.

---

## Funcionalidades
- Cadastro e login de usuários  
- Edição de perfil  
- Criar e excluir posts (drops)  
- Curtir e descurtir publicações  
- Comentar em publicações  
- Encontrar usuários 
- Seguir e deixar de seguir usuários 
- Feed de amigos

## Como Executar o Projeto
1. Coloque os arquivos do projeto em um servidor local (WAMP, XAMPP, Laragon etc.).
2. Crie o banco *droppr*.
3. Importe o arquivo *droppr.sql* (UTF-8) via phpMyAdmin.
4. Configure suas credenciais de banco no arquivo *conexao.php*.
5. Verifique se a pasta *uploads/* tem permissão de escrita.
6. Abra *http://localhost/droppr*.
